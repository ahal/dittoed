# Wellington

A tool to bootstrap your development environment on new machines.


# Installation

Install via `pip`:

    $ pip install wellington

Or `pipx`:

    $ pipx install wellington


# Usage

To install your environment:

    $ well install [ROUTINE]

To update your environment:

    $ well update [ROUTINE]
